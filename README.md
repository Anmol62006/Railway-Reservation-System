# Railway Reservation System

## Overview
Railway Reservation System is a Console-Based Core Java Project developed using Java and File Handling. It allows users to register, login, search trains, check seat availability, book tickets, order food, view booking history, search bookings by PNR, and cancel tickets.

## Features
- User Registration & Login
- Train Search
- Seat Availability
- Ticket Booking
- Multiple Passenger Booking
- Food Ordering
- Unique PNR Generation
- Booking History
- Search by PNR
- Ticket Cancellation
- Change Password

## Technologies Used
- Core Java
- File Handling
- Object-Oriented Programming (OOP)
- Exception Handling
- Arrays

## How to Run
1. Open the project in IntelliJ IDEA or Eclipse.
2. Compile and run the Java file.
3. Register or Login.
4. Use the menu to access different features.
